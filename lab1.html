The point of the first lab is to get to acquainted with the Amazon Alexa Skill Kit. We will go through the basic workflow of deploying a simple Alexa Skill. This involves creating an Amazon Developer account and an Amazon AWS account. At the end of this lab, you should be able to have a single-turn converstaion with your bot using an Alexa-enabled device. For example,

  • User: Alexa, open BOT NAME.
  • Bot: Nice to meet you!

Preparation

  1. Sign up for Amazon AWS. Note that you should to sign up with your @uw.edu account. You will get Free Tier access.
  2. Sign up for Amazon Educate. You will need your account ID which you can find under "my account" from the AWS Admin Panel. This should give you a $100 credit for using AWS. [FAQ]
  3. Create an Amazon Developer account.
  4. Follow instructions from this Github repository.
